Description

This software package developed by the TRMM Office is in GVBOX.  It contains shared C and Perl routines and programs used by the software developed for TRMM Ground Validation purposes.To use the shared routines in this package, include the library, libgv_utils.a, and the header file, gv_utils.h, in your Makefile and/or gv_utilities.plin your Perl program as shown in the Examples Section.

C Function Prototypes

C Function Prototypes are defined in the following header files:
  • gv_file.h. Contains routines for file manipulations.
  • gv_string.h. Contains routines for string manipulations.
  • gv_utilities.h. Contains miscellaneous routines.
  • gv_site.h. Contains GV sites related routines.
  • gv_time.h. Contains time related routines.

Perl Routines

    Perl routines are defined in gv_utilities.pl.

Programs

  •  construct_time_in_seconds. Construct time in seconds.
  •  get_gv_radar_site_info. Get GV radar site information from gv_radar_site_info.data.
  •  get_site_info_from_raw_file. Get GV radar site information from raw radar file.

C Library

    libgv_utils.a

Data File(s)

gv_radar_site_info.data.  Contains central GV radar site information.  Application should use the provided interface routine to access site information for compatibility.

Examples

Using the C Shared routines

           /* construct_time_in_seconds.c
            */
            ...
           #include <gv_utils.h>

           main (int argc, char **argv)
           {
                  int yr, mon, day, hr, min, sec;
                  time_t time_secs;
                ...
                  yr = 1998;
                  mon = 8;
                  day = 1;
                  hr = 1;
                  min = 20;
                  sec = 0;
                  time_secs = gv_utils_construct_time(yr, mon, day, hr, min, sec);
                ...
           }

           # Makefile 

    TOPDIR = /usr/local/trmm/GVBOX/
GV_UTILS_LIBDIR = $(TOPDIR)/lib
GV_UTILS_INCDIR = $(TOPDIR)/include
gv_utils_libs = -L$(GV_UTILS_LIBDIR) -lgv_utils
libs1 =  -L. $(gv_utils_libs)
PROG1 = construct_time_in_seconds
OBJS1 = construct_time_in_seconds.o 
LDFLAGS = -O
CFLAGS = -ansi
CINC = -I. -I$(GV_UTILS_INCDIR)
$(PROG1): $(OBJS1) Makefile
        $(CC) $(LDFLAGS) -o $@ $(OBJS1) $(libs1)
.c.o:  Makefile
        $(CC)  -c $(CFLAGS) $(CINC) $<

Using the Perl routines

    # myprog.pl
    unshift (@INC, "/usr/local/trmm/GVBOX/bin");
do 'gv_utilities.pl';
